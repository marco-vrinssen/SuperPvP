# PvPlus

Quality-of-life PvP enhancements for World of Warcraft.

## Features

- **Auto Spirit Release**: Automates the release process in BGs.\n- **Honor Bar Hiding**: Hides the honor bar when it's not needed.\n- **Queue Timers**: Adds countdown timers for PvP queues.\n- **BG Roster Copy**: Export the battleground roster for external use.

## Installation

1. Download the latest version.
2. Extract the files to your `Interface/AddOns` folder.
3. Restart World of Warcraft.
